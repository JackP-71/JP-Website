TBA - FORM (A C# Windows forms project)

> TBA - FORM

this is a relatively small flashcard creator application.

A screenshot of the revising card window within the project

Synopsis of Project:

This was a project undertaken during my time time at college, where it was my college coursework.

It is a revision aid, allowing the user to create flashcards and add them to decks, which they can then revise from.

Features of project

  • This form is a revision aid, which allows users to sign up and create a variety of flashcards,
  • The user is then able to revise each card by scoring each card, depending on how well they know the contents of the card,
  • The user is also able to adjust any user details if they need to (such as their username or password),
  • The program itself uses databases to contain the data - it creates a database everytime a user signs up. This is so other users are unable to see other's cards without logging in first.

As this project was for coursework in college, i also had to create an extensive analysis of the project as I worked on it, this included the background of the project and the programming techniques used. It also details the testing which occurred during the creation of the program.

DiceTD (A GDscript project)

> DiceTD

A small game made for a college extra curricular.

A screenshot taken partway through a game.

Synopsis of Project:

This was a project undertaken during my time at college, it was completed during an extra curricular called 'Introduction to Game Design'

Features of project

  • The game allows time for the user to place turrets before the next wave of enemies approaches,
  • The game is able to be saved and loaded, so the user will be able to continue where they left off last time they played,
  • The user is able to place turrets on appropriate tiles on the map,
  • As the theme of the game was 'Roll of the Dice' the enemies are represented by dice which grow in power as the number of waves the user has completed increases.
